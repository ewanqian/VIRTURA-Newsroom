# 成员分享发布

这里是 VIRTURA 团队成员创作观念、方法论和工作日志的自我书写与发布地。

## 什么是“成员分享”？

艺术创作不应只存在于作品完成后的展览现场，也不应仅仅依赖外部视角的评判。在这里，我们通过文字记录以下内容：
- **搭建过程的思考**：为什么选择某种技术、某种材料？
- **系统与感知的反思**：创作中那些未完成的、被废弃的或不稳定的尝试。
- **阶段性声明**：作为创作者的自我翻译，为外部理解提供最初的锚点。
