# VIRTURA-Newsroom

VIRTURA-Newsroom 是 VIRTURA 的发布出口。

这里整理研究文章、项目复盘、公告与媒体资料索引，不再承担团队本体、资料总仓或内部治理中心的角色。

## What It Is

这个仓库负责四类内容：

- essays
- project recaps
- notes and announcements
- media references

## What It Is Not

这里不是：

- 团队主入口
- 公共导览前台
- 工具主仓
- 内部管理平台

如果你想认识团队，请看 [VIRTURA-Collective](https://github.com/ewanqian/VIRTURA-Collective)。  
如果你想进入公共导览与档案前台，请看 [VIRTURA-SpacePort](https://github.com/ewanqian/VIRTURA-SpacePort)。  
如果你想看个人入口，请看 [portfolio](https://github.com/ewanqian/portfolio)。  
如果你想看工具主线，请看 [SceneForge](https://github.com/ewanqian/SceneForge)。

## Current Status
