# VIRTURA-Newsroom

VIRTURA-Newsroom 是 VIRTURA 的发布出口。

这里用于整理文章、项目复盘、公告和媒体资料，让外部访客可以从较轻的阅读入口理解我们的工作进展。

## Start Here

如果你是第一次打开这个仓库，建议按下面顺序阅读:

- [Essays](./essays/README.md)
- [Project Recaps](./project-recaps/README.md)
- [Notes And Announcements](./notes/README.md)
- [Media Kit](./media-kit/README.md)

## What Gets Published Here

这里主要承接四类内容:

- 研究与方法文章
- 项目复盘与阶段说明
- 简短公告与更新
- 面向外部的媒体资料

## How It Fits In The Ecosystem

`VIRTURA-Newsroom` 负责“发布”，不负责承载全部身份信息。

- 想认识团队: [VIRTURA-Collective](https://github.com/ewanqian/VIRTURA-Collective)
- 想进入公共导览、知识网络与旧档案入口: [VIRTURA-SpacePort](https://github.com/ewanqian/VIRTURA-SpacePort)
- 想看个人入口: [portfolio](https://github.com/ewanqian/portfolio)
- 想看工具主线: [SceneForge](https://github.com/ewanqian/SceneForge)

## Current Status

当前状态是 `publication-layer transition`。

它正在从过去的混合用途，收束为一个更稳定的轻量发布层。
